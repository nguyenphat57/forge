# Bump Compatibility Wrapper

Use this file only as a thin compatibility wrapper.

Canonical activation stays natural-language first, `python scripts/repo_operator.py bump --workspace <workspace> <version|major|minor|patch>`, and host-native `forge-*` skill discovery.

Preview or apply through `python scripts/prepare_bump.py --workspace <workspace> --bump <version|major|minor|patch>`.

Current version is stated and target version is either explicit or justified by inference.

Do not treat a version edit alone as a release-ready claim.
